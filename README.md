# skill-toolkit

Home of **SkillDeck** — a native macOS cheatsheet and launcher for Claude Code skills and
commands, with a built-in marketplace browser and one-click plugin install.

- **App + docs:** [`SkillDeck/`](SkillDeck/) — see its [README](SkillDeck/README.md) to build and run.
- **Design specs & plans:** [`docs/superpowers/`](docs/superpowers/)
  - v1 (cheatsheet + inject): [spec](docs/superpowers/specs/2026-06-14-skilldeck-design.md) ·
    [plan](docs/superpowers/plans/2026-06-14-skilldeck.md)
  - v2 (marketplace browser + install): [spec](docs/superpowers/specs/2026-06-15-skilldeck-v2-marketplace-design.md) ·
    [plan](docs/superpowers/plans/2026-06-15-skilldeck-v2-marketplace.md)

## Quick start

```bash
cd SkillDeck
swift run SkillDeckApp
```

Requires macOS 15+ and Swift 6.1 / Xcode 16. See [SkillDeck/README.md](SkillDeck/README.md)
for permissions and details.
